## Описание проекта
react-movies — это эмуляция поисковика фильмов и сериалов в интернете по апи [OMDb API](https://www.omdbapi.com)

## Функционал:
- Поиск по названию;
- Фильтрация фильмы, сериалы;
- Обработка ошибок;
